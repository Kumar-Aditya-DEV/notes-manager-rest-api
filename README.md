# Notes Management API

This is a simple REST API for managing notes built using Node.js, Express, and MongoDB.

---

## Live API (Render)

https://notes-manager-rest-api-1.onrender.com

---

## Postman Collection

https://documenter.getpostman.com/view/50839415/2sBXqDs2v1

---

## Features

* Create note
* Get all notes
* Get note by ID
* Update note (PUT & PATCH)
* Delete note
* Bulk create & delete

---

## Tech Stack

* Node.js
* Express.js
* MongoDB (Mongoose)
